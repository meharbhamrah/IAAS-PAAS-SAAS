# IAAS-PAAS-SAAS
# Cloud Service Models (PaaS Levels, IaaS, SaaS)

## PaaS Levels Overview

- **PaaS I** → Only source code upload  
- **PaaS II** → Upload your containers  
- **PaaS III** → Upload git commit + K8s manifest  

---

## Cloud Provider Examples

### PaaS I
- AWS → Elastic Beanstalk  
- GCP → App Engine  
- Azure → Azure App Service  
- OpenStack → Solum  

### PaaS II
- AWS → App Runner  
- GCP → Cloud Run  
- Azure → Azure Container Apps  
- OpenStack → Magnum  
- Rancher  
- HashiCorp  

### PaaS III
- AWS → EKS, Lambda  
- OpenShift on OpenStack
